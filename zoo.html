Zootickoon

Zootickoon est un projet issu du site Frebourg.es qui porte sur 3 languages de programmation : HTML, CSS et PHP

Logo élaboré pour le projet

But du projet :
Créer une interface web de gestion de tickets incident
ainsi qu'un formulaire de connexion, et un site web vitrine du zoo présentant la localisation du zoo et les animaux.


Contexte du projet :
Un zoo du nom de "Zootatoo" prend de l'importance chaque année, et se développe de plus en plus.
Afin de mieux gérer leur SI, les deux directeurs ont décidé de faire appel à une startup d'informatique pour développer un outil de ticketting incident, ainsi que leur interface web vitrine, leur logo et leur slogan.
