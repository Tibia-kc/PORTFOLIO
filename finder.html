Finder

Finder est un projet issu du site Frebourg.es qui porte sur 3 languages de programmation : HTML, CSS, PHP et SQL

But du projet :
Ce projet a pour but de créer des interfaces HTML/CSS/PHP, gérant les bases de données dans l'intérêt de 3 hôtels distincts


Contexte du projet :
Trois hôtels font appel à la même startup afin de créer un outil de gestion de leur base de données, mais demandent d'utiliser chacun une méthode différente.
